Fedora (distribución de Linux)

Fedora es una distribución Linux para propósitos generales basada en RPM, que se caracteriza por ser un sistema estable, mantenido por una comunidad internacional de ingenieros, diseñadores gráficos y usuarios. Cuenta con el respaldo y promoción de Red Hat.

El proyecto incluye software libre y de código abierto, y busca liderar en ese ámbito. Los desarrolladores de Fedora prefieren modificar el código fuente original en vez de aplicar parches específicos, lo que asegura mayor compatibilidad con otras variantes de GNU/Linux. Durante sus primeras seis versiones se llamó Fedora Core. La última versión mencionada es Fedora 16 (2011).

Según DistroWatch, Fedora es la tercera distribución más popular, detrás de Linux Mint y Ubuntu.

Características

Distribución

  • Fedora DVD: contiene todos los paquetes disponibles.
  • Live CD: imágenes de CD/DVD que se pueden instalar o usar desde USB.
  • Imagen mínima: usada para instalación por red (HTTP, FTP, NFS).
  • Imagen de rescate: útil si alguna parte del sistema falla, permite reinstalar desde Internet.

Repositorios

En las primeras 6 versiones había dos repositorios principales: Fedora Core y Fedora Extras. Fedora Core contenía los paquetes esenciales, mientras que Fedora Extras era mantenido por la comunidad.

# Repositorio Descripción
1 Main Incluye paquetes esenciales y con soporte técnico. Cumplen licencias libres.
2 Restricted Incluye software propietario necesario (ej. controladores gráficos ATI/NVIDIA).
3 Universe Gran cantidad de programas mantenidos por la comunidad, sin soporte oficial.
4 Multiverse Software sin soporte por no cumplir licencias libres.

Seguridad

Fedora incorpora SELinux, que permite políticas avanzadas de seguridad como el control de acceso obligatorio (MAC). Aunque inicialmente fue desactivado por su impacto, se activó por defecto desde Fedora Core 3.

También incluye herramientas como «Exec Shield», verificación de buffer en compilación y restricciones de acceso a la memoria del núcleo para evitar ataques como los rootkits.
